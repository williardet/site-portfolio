Projets

API REST en FastAPI

Ce projet consiste en la création d'une API REST complète permettant de gérer un ensemble de Pokémon à partir d’un fichier JSON. J’y ai mis en place une architecture structurée avec des endpoints CRUD, une gestion propre des ressources et une modélisation des données via une dataclass dédiée. L’API intègre également un système de filtrage avancé (types, évolution, statistiques, tri dynamique), ainsi qu’une validation stricte des données grâce à FastAPI et Pydantic. J’ai aussi développé une gestion centralisée des erreurs pour garantir des réponses cohérentes et fiables. Enfin, la documentation automatique générée par FastAPI (Swagger UI / ReDoc) rend l’API intuitive, testable et facile à explorer.

Technologies : Python, FastAPI, JSON
